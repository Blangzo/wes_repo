A Simple Clicker Game

Hi, this is a simple clicker game that I learned how to code and did so in a bit over an hour. It isn't very interesting and doesn't have much depth to it, but it works.

Instructions:

  • Clicking the first button earns you points
  • Clicking the second button upgrades the number of points you receive from clicking the first button - at a price
  • Clicking the third button buys autoclickers. Autoclickers automatically click the first button for you
